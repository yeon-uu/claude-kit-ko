# _career — 커리어 허브

경험블록·JD 분석·갭 계획을 한곳에 모은다. 이력서·면접 준비 때 이 폴더만 열면 되는 상태가 목표.

```
_career\
├── experience-blocks\     ← 프로젝트·대회·동아리당 1파일 (experience-block 스킬이 작성)
│   └── INDEX.md           ← 블록 한 줄 인덱스 (jd-fit 스킬이 검색)
├── jd-analysis\           ← JD 분석 리포트 (jd-fit 스킬이 저장)
├── gap-plan.md            ← 갭 → 채움 수단 누적 계획
└── applications.md        ← 지원·분석 트래커
```

## 운영 규칙

- 경험블록은 프로젝트가 마일스톤(배포·수상·제출·발표)에 닿을 때마다 갱신 — 끝나고 몰아 쓰지 않는다.
- 여기 있는 파일은 면접용 원본(STAR 풀버전 포함)이라 **비공개**. 공개 레포에 올리지 않는다.
- 프로젝트 쪽 `docs/EXPERIENCE.md`에는 공개용 축약본만 (standards/docs-structure.md 참조).
